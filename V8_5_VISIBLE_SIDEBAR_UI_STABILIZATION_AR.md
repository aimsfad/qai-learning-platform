# V8.5 Visible Sidebar + UI Stabilization

هذه النسخة تعالج مشكلة ظهور الشريط الجانبي وتعيد تبسيط CSS بعد أن أصبحت الواجهة غير مستقرة بصرياً.

## ما تم إصلاحه

- إعادة بناء ملف `.streamlit/style.css` من الصفر تقريباً لتقليل التضارب.
- إزالة التراكمات المتضاربة بين الثيم الداكن والفاتح.
- تثبيت الشريط الجانبي الأصلي في Streamlit بخلفية فاتحة وتباين عالٍ.
- جعل محتوى الشريط الجانبي يظهر حتى إذا لم تُحمّل بعض تنسيقات CSS.
- إزالة الاعتماد على طبقات شفافة أو ألوان داكنة كانت تجعل القائمة تبدو فارغة.
- تحسين بطاقات Learning Path وإضافة مسافات داخلية وتنسيق أفضل للوسوم.
- ضبط أعلى الصفحة حتى لا يختفي العنوان خلف شريط Streamlit.

## ملاحظات

هذه نسخة استقرار بصري. لم تغيّر قاعدة البيانات ولا منطق تسجيل الدخول ولا منطق الاختبارات ولا محرك الذكاء الاصطناعي.
