# V6.5 Modular Lesson Layout Update

## What changed
- Reorganized each learning module into 5 clear tabs:
  - Overview
  - Concept
  - Code and output
  - Visual and video
  - Check and reflect
- Reduced overload from the old single long lesson view.
- Kept the learner in a structured flow instead of forcing them to decode a dense image.
- Split the media explanation into separate blocks:
  - Diagram
  - Code map
  - Micro-video
  - What to notice
- Preserved the existing AI support, progress tracking, and reflection logging.

## Goal
This version is intended to make the learning path clearer, smoother, and less ambiguous for beginner learners in introductory quantum programming.
