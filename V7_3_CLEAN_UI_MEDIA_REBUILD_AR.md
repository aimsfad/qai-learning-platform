# V7.3 Clean UI + Media Rebuild

هذا التحديث يعالج المشاكل التي ظهرت في النسخة السابقة:

- إزالة تضارب CSS الذي جعل القائمة اليسرى والنصوص غير واضحة.
- استبدال رمز الشعار السابق بلوغو نصي واضح: QAI.
- تخفيف اللون الليلي وتحويل الواجهة إلى ثيم احترافي فاتح/ناعم.
- إظهار الوسائط داخل الدرس بطريقة مجزأة بدل صورة واحدة مزدحمة.
- جعل تبويب Visual and video أول تبويب في الدرس.
- تقسيم الشرح إلى: concept diagram، tiny Qiskit code، read order، before measurement، after measurement، micro-video.
- إبقاء الصورة الكاملة اختيارية داخل expander فقط، وليست العنصر الأساسي.
- إعادة ترميز فيديوهات MP4 بصيغة H.264/yuv420p لتحسين تشغيلها في المتصفح وStreamlit Cloud.
- عرض حجم ملف الفيديو داخل التطبيق للتأكد أن الفيديو مستدعى فعليًا.
