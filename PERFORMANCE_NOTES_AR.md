# تحسينات الأداء في v3.8

هذه النسخة أخف من النسخ السابقة لأنها:

1. لا تنشئ/تفحص جداول قاعدة البيانات في كل rerun؛ تم تخزين `init_db()` مؤقتًا.
2. لا تحمل كل سجلات الذكاء الاصطناعي في لوحة المقيم الرئيسية، بل تعرض إحصاءات مجمعة فقط.
3. صفحة Feedback Logs تحمل آخر 50/100/200/500 سجل حسب اختيار المقيم بدل تحميل كل السجلات.
4. صفحة Event Logs تحمل آخر عدد محدد من الأحداث فقط.
5. صفحة Results Export لا تبني ملف Excel الضخم تلقائيًا عند فتح الصفحة؛ يتم ذلك فقط عند الضغط على Prepare full export workbook.
6. أزيل الاعتماد على `st.bar_chart` في الصفحات الحساسة واستُخدم عرض أخف وأكثر استقرارًا.

## ملاحظات للنشر على Streamlit Cloud

- استخدمي PostgreSQL Cloud عبر `DATABASE_URL`.
- لا ترفعي `.streamlit/secrets.toml` إلى GitHub.
- لا ترفعي قاعدة SQLite المحلية.
- عند كثرة الطلبة أو سجلات AI، استعملي صفحات التحليل والتصدير بدل عرض الجداول الخام كاملة.

## إذا بقيت المنصة ثقيلة

- قللي عدد الدروس أو النصوص الطويلة داخل الدروس.
- اجعلي ردود LLM قصيرة عبر التعليمات: 120-180 كلمة.
- لا تضغطي على Results Export إلا عند الحاجة.
- في Feedback Logs اختاري 50 أو 100 row فقط.
