# تحديث عرض حسابات التسجيل للمقيم

## الهدف

أضيفت صفحة جديدة في واجهة المقيم باسم **Registration Accounts** حتى يستطيع المقيم/الباحث الاطلاع على معلومات التسجيل الأساسية لحسابات المشاركين، ومساعدة الطلبة الذين نسوا البريد أو participant code أو لم يستطيعوا الدخول.

## ماذا تعرض الصفحة؟

تعرض الصفحة:

- participant code
- الاسم الكامل
- البريد الإلكتروني المستخدم في التسجيل
- المؤسسة
- المستوى الأكاديمي
- مستوى Python السابق
- مستوى معرفة quantum السابق
- تاريخ إنشاء الحساب
- آخر تسجيل دخول
- حالة الحساب active / inactive

كما تعرض مؤشرات سريعة:

- عدد الحسابات المسجلة
- عدد الحسابات النشطة
- عدد الحسابات التي لا تحتوي على بريد
- عدد الحسابات التي سجلت الدخول مرة واحدة على الأقل

## ميزات إضافية

- البحث بالاسم أو البريد أو participant code أو المؤسسة.
- فلترة الحسابات النشطة فقط.
- فلترة الحسابات التي لا تحتوي على بريد.
- فلترة الحسابات التي لم تسجل الدخول بعد.
- تحميل قائمة حسابات التسجيل بصيغة CSV.

## حماية الخصوصية

الصفحة لا تعرض:

- كلمات المرور
- password hash
- password reset tokens
- أي مفاتيح سرية

لذلك إذا نسي الطالب كلمة المرور، يجب استعمال خاصية **Forgot password** بدل محاولة معرفة كلمة المرور القديمة.
