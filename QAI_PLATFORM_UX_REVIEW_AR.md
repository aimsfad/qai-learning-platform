# مراجعة UX وتعليمية لمنصة QAI

## التشخيص
المشكلة الأساسية ليست تقنية فقط، بل هي وضوح مسار الطالب. الطالب يحتاج أن يرى دائمًا: أين أنا الآن؟ ما الخطوة التالية؟ ولماذا لا يظهر لي الاختبار البعدي؟

## ما تم تحسينه في الكود
1. إضافة خريطة تقدم من 6 مراحل.
2. إضافة رسالة Next action في صفحة الطالب.
3. فتح الاختبار البعدي بعد إنجاز الحد الأدنى المطلوب للدراسة: نشاط تعلم واحد + تفاعل واحد مع AI Tutor.
4. إضافة prompts جاهزة في AI Tutor حتى لا يبدأ الطالب من صفحة فارغة.
5. إضافة mini task قبل طلب مساعدة الذكاء الاصطناعي لتقليل الاعتماد الكلي على AI.
6. تحسين رسالة حفظ participant code.
7. إضافة شريط تقدم في صفحة الدروس.

## تحسينات مستقبلية مقترحة
- إضافة فيديو قصير أو GIF تعريفي مدته 30 ثانية في صفحة Student Home.
- إضافة مثال visual صغير للـ counts داخل درس Hadamard.
- إضافة زر Copy participant code باستخدام streamlit-copy-to-clipboard أو st.components لاحقًا.
- إضافة صفحة FAQ قصيرة: لماذا لا يظهر post-test؟ ماذا أفعل إذا نسيت الكود؟ كيف أسأل AI Tutor؟
- إضافة اختيار لغة مفضل أثناء التسجيل وتخزينه في قاعدة البيانات.
